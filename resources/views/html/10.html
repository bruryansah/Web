Visual Studio Code

Visual Studio Code adalah Aplikasi editor teks yang dikembangkan oleh Microsft yang dapat di gunakan disemua bahasa Programan yang ada tanpa perlu berganti aplikasi editor, serta dapat dijalankan di berbagai platfrom operating sistem (OS).

kelas 10 akan diajarkan menggunakan bahasa pemprogram Python karena Bahasa ini memiliki Sintask-Sintaks yang cukup sederhana dan Gamapang dimengerti. Dan Juga akan diajarkan tenteng HTML dan CSS.

MySQL

MySQL adalah Sebuah Program database Server yang mampu menerima dan mengirimkan datanya sangat cepat, multi user serta menggunakan perintah dasar SQL (Structured Query Language).

kelas 10 akan diajarkan untuk Memahami konsep dasar Penggunaan perintah Data Definition Langangue serta mampu memanfaatkan Command Line (Client) dan juga penggunaan query SQL yang dipadukan dengan Bahasa Pemprograman tertentu.
